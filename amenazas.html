Amenazas Comunes

Conoce los principales ataques cibernéticos y cómo identificarlos antes de ser víctima.

Phishing

¿Qué es? Correos o mensajes fraudulentos que simulan ser de empresas legítimas para robar información personal.

El 90% de brechas de datos comienzan con phishing.

Ransomware

¿Qué es? Software malicioso que cifra tus archivos y exige un rescate (generalmente en criptomonedas) para desbloquearlos.

Los ataques aumentaron 150% en 2023.

Malware

¿Qué es? Software malicioso diseñado para dañar, infiltrar o robar información de un sistema sin el consentimiento del usuario.

560,000 nuevas piezas de malware detectadas diariamente.

Ataques DDoS

¿Qué es? Ataque de Denegación de Servicio Distribuido que satura un servidor con tráfico hasta dejarlo inoperativo.

Pueden costar hasta $2.5 millones por hora.

Casos Reales de Ataques

Ataque a Colonial Pipeline (2021)

Ransomware DarkSide paralizó el mayor oleoducto de combustible de EE.UU., causando escasez de gasolina en la costa este.

Rescate pagado: $4.4 millones en Bitcoin
Duración: 6 días de paralización
Lección: La infraestructura crítica es vulnerable

Ataque al Sistema de Salud de Irlanda (2021)

Ransomware Conti comprometió el sistema de salud completo del país, cancelando cirugías y servicios médicos durante semanas.

Costo estimado: $600 millones
Datos robados: 700GB de información sensible
Lección: Los servicios esenciales deben priorizar la seguridad

Phishing Masivo Amazon (2022)

Campaña de phishing que suplantó a Amazon durante la temporada navideña, robando credenciales y datos bancarios de miles de usuarios.

Víctimas: Más de 100,000 personas
Método: Correos falsos de "problema con tu pedido"
Lección: Siempre verificar la URL y el remitente

Phishing - Información Detallada
¿Cómo funciona el Phishing?
  1. Engaño: Recibes un correo que parece legítimo (banco, Amazon, etc.)
  2. Urgencia: El mensaje crea presión ("Tu cuenta será suspendida")
  3. Acción: Te piden hacer clic en un enlace falso
  4. Robo: Introduces tus datos en una página falsa
Señales de Alerta:
  • Errores ortográficos o gramática extraña
  • Remitente con dominio sospechoso
  • Solicitan información confidencial
  • Urgencia o amenazas injustificadas
Consejo: NUNCA hagas clic en enlaces de correos no solicitados. Ve directamente al sitio web oficial.
Ransomware - Información Detallada
¿Cómo funciona el Ransomware?
  1. Infección: Entra por correo, descarga o vulnerabilidad
  2. Cifrado: Encripta tus archivos con algoritmos potentes
  3. Nota de rescate: Muestra un mensaje exigiendo pago
  4. Amenaza: Si no pagas, pierdes tus archivos para siempre
Tipos de Ransomware:
  • Crypto-ransomware: Cifra archivos específicos
  • Locker-ransomware: Bloquea el acceso completo al sistema
  • Scareware: Finge ser legítimo (antivirus falso)
  • Doxware: Amenaza con publicar información privada
Importante: NUNCA pagues el rescate. No hay garantía de recuperar tus archivos y financias el crimen.
Cómo Protegerte:
  • ✅ Haz copias de seguridad regularmente (regla 3-2-1)
  • ✅ Mantén el software actualizado
  • ✅ Usa antivirus con protección en tiempo real
  • ✅ No abras archivos adjuntos sospechosos
Malware - Información Detallada
¿Qué es el Malware?

"Malware" es un término general para cualquier software malicioso diseñado para dañar, explotar o comprometer sistemas, redes o dispositivos.

Tipos Principales de Malware:
Virus

Se adhiere a archivos legítimos y se replica al ejecutarse.

Gusanos (Worms)

Se propagan automáticamente sin necesidad de un archivo host.

Troyanos

Se disfrazan de software legítimo para engañarte.

Spyware

Espía tu actividad y roba información personal.

Adware

Muestra publicidad no deseada de forma invasiva.

Rootkits

Se oculta en el sistema operativo para evitar detección.

Cómo Detectar Malware:
  • El dispositivo se vuelve lento de repente
  • Aparecen ventanas emergentes constantemente
  • Programas desconocidos se ejecutan solos
  • El antivirus se desactiva misteriosamente
  • Cambios no autorizados en archivos o configuración
Prevención:
  • 🛡️ Instala un antivirus confiable y mantenlo actualizado
  • 🔄 Actualiza el sistema operativo regularmente
  • 🚫 No descargues software de fuentes no oficiales
  • 🔍 Escanea archivos antes de abrirlos
  • 🌐 Usa un navegador con protección integrada
Ataques DDoS - Información Detallada
¿Qué es un Ataque DDoS?

DDoS significa "Distributed Denial of Service" (Denegación de Servicio Distribuida). Es un ataque que busca hacer que un sitio web o servicio sea inaccesible sobrecargándolo con tráfico masivo.

¿Cómo Funciona?
  1. Botnet: El atacante controla miles de dispositivos infectados (zombies)
  2. Coordinación: Todos los dispositivos atacan el mismo objetivo simultáneamente
  3. Sobrecarga: El servidor no puede manejar tantas peticiones
  4. Caída: El servicio se vuelve inaccesible para usuarios legítimos
Volumétricos

Saturan el ancho de banda con tráfico masivo

Protocolo

Explotan debilidades en protocolos de red

Aplicación

Atacan la capa de aplicación web

Impacto de un Ataque DDoS:
  • 💰 Pérdidas financieras: $20,000 - $40,000 por hora
  • 📉 Daño reputacional: Pérdida de confianza de clientes
  • ⏱️ Tiempo de inactividad: Horas o días sin servicio
  • 🔒 Cortina de humo: A veces oculta otros ataques
Protección Contra DDoS:
  • ☁️ Usa servicios de mitigación (Cloudflare, AWS Shield)
  • 🔄 Implementa balanceo de carga
  • 📊 Monitorea el tráfico en tiempo real
  • 🚦 Configura límites de tasa (rate limiting)
  • 🛡️ Ten un plan de respuesta a incidentes
Dato curioso: El ataque DDoS más grande registrado alcanzó 3.47 Tbps (terabits por segundo) en 2024, equivalente a descargar toda Wikipedia en menos de 1 segundo.
